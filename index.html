Cristian's Cheat Sheet

Table Tag

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Media Elements

Tag Name Description
<img> Image Self closing tag with attributes src and alt to describe image when it is not loaded
<video> Video Video element with attributes src, type, width, height, controls
<figure> Figure Represent self-contained content as images or videos with their respective tags
<figcaption> Figure Caption Caption that can be added underneath the media added

CSS Selectors

Selector Description
#id Select the element by id, this is the most specific and unique
.class Select all elements with that class and can be uses multiple times
element.class Select element with that class ex: h1.red
element Select an element by their tag as, h1, body...
element,element Select the elements you applied ex: h1, p
* Select all elements
: Pseudo class (:hover, :dir...) let you apply style to an element

Common CSS Properties

Propertie Description Values
font-family: Sets the font family for an element; Type the name of the font/s in single/double quotes;
font-size: Sets the size of the font for an element; px, %, em, small, larger, x-large
font-weight: Sets the weight or thickness of the font; inherits(inherits the value from its parent element), lighter, normal, bold, bolder or by numbers between 100(thinest) and 900(boldest)
text-align: Sets the alignment of text inside an element; left, center, right, justify
color: Specifies the color of text in an element; could be specifies with text or hex #000000, rgb(0,255,0), hsl(360,100%,100%), rgba(0,255,0,0.3), hsla(360,100%,100%,0.5)
background-color: Specifies background color of the element; Same as color
opacity: Defines the transparency of an element; Values range from 0 (transparent) to 1 (opaque)
background-image: Set the element's background image; url(''), height,
!important Add more importance to a property/value, override This is a value
list-style: Specifies the markers for the item; dise, circle, square, kannada...
border: Apply borders to a box; ex: 3px solid coral (width, style and color)
border-radius: Modify borders of a box; px, %
padding: Space in between the content and the border, could be specifies as top, right, bottom, left single value or multiple values (6px 3px 4px 79px)
margin: Space outside of the box, could be specifies as top, right, bottom, left Same as padding or (0 auto)
